Detailed Description

Implements Four In Row game logic on a 6x7 grid.

This class manages the Four In Row game board where players drop pieces into columns. Pieces fall to the lowest available position in the chosen column. The game ends when a player gets four pieces in a row horizontally, vertically, or diagonally.
