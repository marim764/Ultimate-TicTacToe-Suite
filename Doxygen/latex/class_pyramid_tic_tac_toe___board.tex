\doxysubsection{Detailed Description}
Represents the pyramid-\/shaped Tic-\/\+Tac-\/\+Toe game board. 

This class implements the rules, board layout, and win/draw conditions for the Pyramid Tic-\/\+Tac-\/\+Toe game. It inherits from \doxylink{class_board}{Board$<$char$>$}.
