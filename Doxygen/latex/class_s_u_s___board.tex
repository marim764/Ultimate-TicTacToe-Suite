\doxysubsection{Detailed Description}
Represents the game board for the SUS (S-\/\+U-\/S) game. 

The game is played on a 3x3 grid where players take turns placing \textquotesingle{}S\textquotesingle{} (\doxylink{class_player}{Player} 1) and \textquotesingle{}U\textquotesingle{} (\doxylink{class_player}{Player} 2). Points are scored by creating "{}\+S-\/\+U-\/\+S"{} sequences horizontally, vertically, or diagonally. The player with the most points when the board is full wins.
