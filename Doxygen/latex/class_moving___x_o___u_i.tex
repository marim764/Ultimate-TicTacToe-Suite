Gets a move from the player. 

Gets a move from the player (human input or computer random move).

For humans\+:
\begin{DoxyItemize}
\item Enter source x,y and destination x,y.
\item Validates ownership, empty destination, and orthogonal adjacency.
\end{DoxyItemize}

For computer\+:
\begin{DoxyItemize}
\item Randomly selects a movable token and a valid destination.
\end{DoxyItemize}
