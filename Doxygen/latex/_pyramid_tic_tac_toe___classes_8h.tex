\doxysubsection{Detailed Description}
Defines classes for the Pyramid Tic-\/\+Tac-\/\+Toe game. 

Represents a Tic-\/\+Tac-\/\+Toe variant played on a pyramid-\/shaped board. The pyramid consists of\+:
\begin{DoxyItemize}
\item 1 cell at the top (row 0)
\item 3 cells in the middle (row 1)
\item 5 cells at the base (row 2) 
\end{DoxyItemize}
